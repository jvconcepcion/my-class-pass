## üöÄ Getting Started

1. **Clone the repository:**
   ```sh
   git clone https://github.com/your-username/my-class-pass.git
   cd my-class-pass 
   ```

2. **Install dependencies:**
   ```sh
   npm install
   # or
   yarn install
   ```

3. **Run the development server:**
   ```sh
   npm run dev
   # or
   yarn dev
   ```
   The app should now be running at `http://localhost:3000`.
